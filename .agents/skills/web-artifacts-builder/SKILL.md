# Web Artifacts Builder Skill

Use this skill when an agent creates standalone browser artifacts.

## Instructions
1. Produce complete files with no missing asset references.
2. Keep scripts scoped and avoid unsafe remote execution.
3. Include accessible labels, responsive layout, and clear empty/error states.
4. Validate the artifact in a browser or with a static check before marking work complete.
