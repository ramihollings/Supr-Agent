# Frontend Design Skill

Use this skill for Supr UI work that needs a complete, user-ready interface.

## Instructions
1. Build the working experience first; avoid placeholder-only screens.
2. Keep operational tools dense, readable, and permission-aware.
3. Make supervisor decisions visible with status, rationale, and next action.
4. Verify text fit, responsive behavior, and disabled/error/loading states.
